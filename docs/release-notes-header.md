> **First-time install:** Download the first asset below &mdash; `lp-ranger-<version>.tar.gz` (the large ~10 MB file; **not** "Source code" and **not** the `.sha256`) &mdash; then follow the **Production** install instructions: https://github.com/nottoseethesun/lp-ranger#install
>
> **Updating an existing install?** Do **NOT** follow the Install instructions &mdash; they assume a clean directory and would create a separate versioned install rather than upgrading your current one. Follow the **Update** section instead, which preserves your wallet, managed positions, and any custom overrides while replacing only the shipped code and shipped defaults: https://github.com/nottoseethesun/lp-ranger#update
